# 误差先验建模实验运行指南

## 快速开始

### 1. 运行基础误差先验建模实验

直接运行主文件，会自动执行完整的实验流程：

```bash
python3 error-prior-modeling.py
```

**实验流程包括：**
- 步骤1: 初始化误差先验模型（工作空间 300×300×200mm）
- 步骤2: 生成50个标定点
- 步骤3: 模拟测量过程（每个点重复测量10次）
- 步骤4: 拟合RBF插值模型
- 步骤5: 计算置信度图（256×256像素）
- 步骤6: 分析空间非均匀性
- 步骤7: 模型验证（留出法，80%训练，20%验证）
- 步骤8: 生成误差数据表

## 参数说明

### ErrorPriorModel 主要参数

```python
model = ErrorPriorModel(
    workspace_size=(300, 300, 200),  # 工作空间大小 (mm)
    length_scale_factor=1.5           # RBF长度尺度参数因子
)
```

### 标定点生成参数

```python
calibration_points = model.generate_calibration_points(
    n_points=50,  # 标定点数量，建议>=50
    seed=42       # 随机种子，用于重现结果
)
```

### 测量模拟参数

```python
sys_errors, rand_std = model.simulate_measurements(
    n_repeats=10,              # 每个标定点的重复测量次数
    sensor_noise_std=0.5,     # 传感器噪声标准差 (mm)
    transform_error_std=1.0,  # 坐标变换误差标准差 (mm)
    seed=42                   # 随机种子
)
```

### 置信度图参数

```python
confidence_map, sigma_2d, tau = model.compute_confidence_map(
    image_size=(256, 256),    # 超声图像尺寸
    pixel_spacing=(1.0, 1.0)  # 像素间距 (mm/pixel)
)
```

## 自定义实验

### 示例：修改标定点数量

```python
from error_prior_modeling import ErrorPriorModel, run_full_experiment

# 创建模型
model = ErrorPriorModel(workspace_size=(300, 300, 200))

# 生成更多标定点
calibration_points = model.generate_calibration_points(n_points=100, seed=42)

# 模拟测量
sys_errors, rand_std = model.simulate_measurements(n_repeats=20, seed=42)

# 拟合模型
model.fit_rbf_model()

# 计算置信度图
confidence_map, sigma_2d, tau = model.compute_confidence_map(image_size=(256, 256))
```

### 示例：预测特定位置的误差

```python
# 预测任意位置的误差
query_points = np.array([
    [150, 150, 100],  # 工作空间中心
    [50, 50, 50],     # 边缘区域
    [250, 250, 150]   # 另一个边缘区域
])

systematic_error, random_error_std = model.predict_error(query_points)
print("系统误差:", systematic_error)
print("随机误差标准差:", random_error_std)
```

## 输出结果说明

### 验证指标

- **相关系数R**: 预测值与真实值的相关性，目标 > 0.85
- **RMSE**: 均方根误差，系统误差预测目标 < 0.5 mm

### 空间非均匀性分析

- **中心区域**: 距磁源 < 120mm，高置信度区域
- **边缘区域**: 距磁源 ≥ 120mm，低置信度区域

## 注意事项

1. **依赖安装**: 确保已安装所有依赖（见 requirements.txt）
2. **随机种子**: 使用固定种子（seed=42）可重现实验结果
3. **工作空间**: 默认工作空间为 300×300×200mm，可根据实际设备调整
4. **标定点数量**: 建议至少50个标定点，更多标定点可提高模型精度
